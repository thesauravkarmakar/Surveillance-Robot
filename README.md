# Surveillance-Robot
Robots are changing the world. Robots are great. Robots with eyes are even better!
This surveillance robot can be controlled from a remote location and capture live video footages.

Main components are:\
Raspberry Pi B+\
60 RPM Motors \
L239D Motor Driver
